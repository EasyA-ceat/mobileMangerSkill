# 手机控制技能包技术方案（MVP版本）
## 一、整体架构
```
┌─────────────────┐     ┌──────────────────┐     ┌───────────────┐
│ OpenClaw 技能层 │────▶│ 能力封装接口层   │────▶│ 设备控制层     │
└─────────────────┘     └──────────────────┘     └───────────────┘
      ▲                        ▲                        ▲
      │                        │                        │
   技能配置                业务逻辑封装               ADB/Scrcpy
参数校验/权限控制       触控/按键/截屏/执行脚本    底层系统交互
```
## 二、技术选型
| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 开发语言 | Python 3.10+ | 符合用户常用技术栈，适配OpenClaw技能生态 |
| 接口层 | FastAPI + Uvicorn | 轻量高性能，符合OpenClaw技能接口规范 |
| 安卓控制 | ADB + pure-python-adb封装 | 官方原生工具，兼容性最好 |
| 投屏/低延迟控制 | Scrcpy v2.0+ | 开源成熟方案，支持硬件编码，延迟最低 |
| OCR能力 | Tesseract OCR 5.x | 开源免费，支持中文识别，准确率满足需求 |
| 屏幕处理 | OpenCV-Python | 实现截图比对、坐标匹配等功能 |
## 三、核心模块设计
### 1. 设备管理模块 (device_manager.py)
- 功能：自动扫描USB/局域网ADB设备、设备连接/断开/状态管理、多设备并发支持
- 对外接口：list_devices()、connect_device()、disconnect_device()、get_device_status()
### 2. ADB控制模块 (adb_client.py)
- 功能：封装所有ADB操作接口，包括点击、滑动、输入文字、按键控制、App管理、文件传输
- 对外接口：click()、swipe()、input_text()、press_key()、start_app()、stop_app()、list_apps()
### 3. 屏幕交互模块 (screen.py)
- 功能：截屏、OCR文字识别、文字坐标匹配、图片比对
- 对外接口：screenshot()、ocr_text()、find_text_position()、click_by_text()
### 4. 脚本执行模块 (script_runner.py)
- 功能：解析JSON格式操作脚本，顺序执行操作序列，支持延时、条件判断
- 脚本格式：
```json
[
  {"action": "key", "params": {"key": "home"}},
  {"action": "click", "params": {"target": "微信"}},
  {"action": "sleep", "params": {"seconds": 2}},
  {"action": "swipe", "params": {"start": "100,2000", "end": "100,500"}}
]
```
## 四、项目结构（符合OpenClaw技能规范）
```
phone-control-skill/
├── SKILL.md           # OpenClaw技能说明文件
├── manifest.json      # 技能元信息声明
├── main.py            # FastAPI接口入口
├── src/
│   ├── adb_client.py  # ADB操作封装
│   ├── scrcpy_client.py # Scrcpy投屏控制
│   ├── device_manager.py # 设备管理
│   ├── screen.py      # 屏幕交互与OCR
│   └── script_runner.py # 脚本执行引擎
├── requirements.txt   # Python依赖声明
├── install.sh         # 自动安装脚本（含系统依赖）
└── docs/              # 开发文档（当前目录）
```
## 五、性能指标要求
- 指令响应延迟：≤500ms
- 截屏耗时：≤2s
- OCR识别准确率：≥90%
- 连续运行稳定性：≥24小时无崩溃
