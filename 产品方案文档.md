## 一、产品定位

### 1.1 产品愿景

**成为开发者首选的移动端自动化与控制工具**，通过API优先的设计理念，让移动端自动化触手可及。

### 1.2 核心价值主张

| 价值维度 | 具体描述 |
|---------|---------|
| **技术优势** | ADB+Scrcpy双引擎，兼顾控制能力与低延迟投屏 |
| **易用性** | 内置OCR智能交互，无需记忆坐标 |
| **开放性** | 完全开源（Apache 2.0），可定制、可扩展 |
| **集成性** | API优先设计，无缝接入OpenClaw生态 |

### 1.3 差异化定位

| 对比维度 | 本产品 | ToDesk/向日葵 | Scrcpy | Appium |
|---------|-------|--------------|--------|--------|
| **目标用户** | 开发者/自动化 | 普通用户/办公 | 技术极客 | QA测试 |
| **核心能力** | API+自动化 | 远程协作 | 纯投屏 | 测试框架 |
| **易用性** | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐ |
| **自动化** | ⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| **开源免费** | ✅ 完全免费 | ❌ 付费版 | ✅ 免费 | ✅ 免费 |

### 1.4 产品Slogan

> **"用代码控制手机，释放双手"**

### 4.2 技术选型

| 层级 | 技术选型 | 选型理由 |
|-----|---------|---------|
| **开发语言** | Python 3.10+ | 生态丰富、快速开发、符合OpenClaw技能规范 |
| **Web框架** | FastAPI + Uvicorn | 高性能、异步支持、自动生成API文档 |
| **前端框架** | Vue 3 + Element Plus | 组件丰富、开发效率高、社区活跃 |
| **Android控制** | ADB + pure-python-adb | 官方原生、兼容性最好 |
| **投屏方案** | Scrcpy v2.0+ | 开源成熟、硬件编码、延迟<50ms |
| **OCR引擎** | Tesseract OCR 5.x | 开源免费、支持中文、准确率满足需求 |
| **图像匹配** | OpenCV-Python | 功能强大、模板匹配算法成熟 |
| **任务队列** | Celery + Redis | 分布式任务、支持定时调度 |
| **数据存储** | SQLite（MVP）→ PostgreSQL | 轻量易部署 → 企业级支持 |

### 4.3 核心模块设计

#### 模块1：设备管理器 (DeviceManager)

**职责**：设备发现、连接管理、状态监控

```python
class DeviceManager:
    def list_devices() -> List[Device]  # 扫描所有ADB设备
    def connect_device(device_id: str) -> bool  # 连接指定设备
    def disconnect_device(device_id: str) -> bool  # 断开设备
    def get_device_status(device_id: str) -> DeviceStatus  # 获取设备状态
```

#### 模块2：脚本引擎 (ScriptEngine)

**职责**：脚本解析、执行控制、异常处理

```python
class ScriptEngine:
    def load_script(script_path: str) -> Script  # 加载脚本
    def execute_script(script: Script, device_id: str) -> ExecutionResult  # 执行脚本
    def record_action(action: Action) -> None  # 录制操作
    def generate_script() -> Script  # 生成脚本
```

#### 模块3：元素定位器 (ElementLocator)

**职责**：多策略元素定位

```python
class ElementLocator:
    def locate_by_text(text: str) -> Coordinate  # OCR文字定位
    def locate_by_image(template: Image) -> Coordinate  # 图像匹配定位
    def locate_by_ui_selector(selector: str) -> Coordinate  # UI树定位
```

| **设备连接成功率** | 成功连接设备数 / 尝试连接数 | 日 |
| **脚本执行成功率** | 成功执行脚本数 / 总执行次数 | 日 |
| **OCR识别准确率** | 人工校验正确数 / 识别总数 | 周 |
| **平均响应延迟** | 操作指令响应时间平均值 | 日 |
| **Crash率** | Crash次数 / 活跃用户数 | 日 |
